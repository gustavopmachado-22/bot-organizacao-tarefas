# 🤖 Bot de Organização de Tarefas (CLI)

Projeto prático desenvolvido para a disciplina de **Prática Profissional em ADS**. O objetivo é criar um assistente simples para gestão de pendências diárias utilizando Python.

## 🚀 Funcionalidades Atuais
- [x] **Adicionar Tarefas:** Cadastro de novas atividades com persistência.
- [x] **Listar Tarefas:** Visualização organizada das pendências gravadas.
- [ ] **Concluir Tarefas:** (Planejado para a iteração 2).
- [ ] **Remover Tarefas:** (Planejado para a iteração 2).

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** [Python 3.x](https://www.python.org)
- **Persistência de Dados:** JSON (JavaScript Object Notation)
- **Versionamento:** Git & GitHub

## 📂 Como rodar o projeto
1. Certifique-se de ter o Python instalado.
2. Baixe o arquivo `bot.py` deste repositório.
3. No terminal/prompt de comando, execute: `python bot.py`
